# 🚀 SmartPlan AI: Multi-Agent Travel Orchestrator

SmartPlan AI is a premium, state-of-the-art travel planning platform powered by a neural graph of specialized AI agents. Using **LangGraph** and **Groq**, it orchestrates complex research, logistical planning, and budget optimization to deliver professional, tailored travel dossiers in seconds.

![SmartPlan AI Preview](https://via.placeholder.com/1200x600?text=SmartPlan+AI+Premium+Interface)

## ✨ Features

-   **🧠 Multi-Agent Orchestration**: A neural graph of specialized agents (Context Engine, Global Researcher, Logistics Planner, and Budget Consultant) working in parallel.
-   **⚡ Real-time Pipeline Visualization**: Watch the AI "think" through a beautiful, live-streaming agent pipeline.
-   **📅 Day-by-Day Itineraries**: Detailed, high-end plans tailored to user preferences and duration.
-   **💰 Smart Budgeting**: Itemized cost breakdowns generated by a dedicated financial advisor agent.
-   **🎨 Premium UI/UX**: A modern, responsive SaaS aesthetic built with React, Vite, and Lucide icons.
-   **📡 Live Streaming (SSE)**: Real-time progress updates delivered via Server-Sent Events.

## 🛠️ Tech Stack

-   **Backend**: Python, FastAPI, LangGraph, LangChain, Groq LLMs.
-   **Frontend**: React, Vite, TailwindCSS (Vanilla CSS Design System), Lucide React.
-   **Orchestration**: StateGraph (LangGraph) for multi-agent coordination.
-   **Data Flow**: SSE (Server-Sent Events) for real-time progress streaming.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/TanishqBhosle/multi-agent-smartplaner.git
cd multi-agent-smartplaner
```

### 2. Backend Setup
```bash
# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Create .env file and add your keys
echo "GROQ_API_KEY=your_key_here" > .env

# Run server
python main.py
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Deployment

### Backend (Render/Railway)
-   **Runtime**: Python 3
-   **Build Command**: `pip install -r requirements.txt`
-   **Start Command**: `uvicorn main:app --host 0.0.0.0 --port $PORT`
-   **Env Vars**: Add `GROQ_API_KEY`.

### Frontend (Vercel)
-   **Framework**: Vite
-   **Root Directory**: `frontend`
-   **Env Vars**: Add `VITE_API_URL` pointing to your deployed backend.
